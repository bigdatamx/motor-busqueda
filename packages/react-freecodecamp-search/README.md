# react-freecodecamp-search

## Install

`npm install react-freecodecamp-search`
-or-
`yarn add react-freecodecamp-search`

## Usage

```jsx
import FCCSearchBar from 'react-freecodecamp-search';

// in your component
return (
  <FCCSearchBar
    placeholder='This is optional'
  />
);
```

Then search for something
